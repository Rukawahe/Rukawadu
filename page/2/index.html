const指针的理解

发表于 2019-09-03 更新于 2019-11-06 分类于 c++

这几天在学算法竞赛，突然常常碰见类似这样的函数体(const vector<int>& A),本来我一开始不明白为什么要用& ,我也不太清楚const有什么作用。后来去查了些资料，发现const关键字是真的挺常用的，而后的& 也是和const相关的。

以下就来举例出常见的几个const用法

const T

定义一个常量，声明的同时必须进行初始化，一旦声明之后该值就不能改变。

const T*

指向常量的指针，不能用于改变其所指向对象的值。

const int*和int *const的区别

把指针定义未常量就是常量指针，也就是int *const类型，声明的同时必须进行初始化

const T&

对常量的引用，称为常量引用，不能修改其绑定的值。

搭建个人博客

发表于 2019-09-01 更新于 2019-11-06 分类于 前端

前言

为什么不使用一些知名博客网站写博客？

虽然市场上有很多优秀的平台，例如csdn、博客园等优秀平台，但是本人使用github开源网站频数较多，所以采用github部署博客的形式。其次，github搭建博客，整个博客的设计可以自己设计，其中涉及到大量的前端知识和其他知识，这是个学以致用的方式。

工具的选择

Atom编辑器+git本地+github远端+Hexo博客框架
(本人现在使用vscode进行文字编辑，相应的markdown语法插件vscode早早配置好了…真香)

现在网上编辑文字的语言仅仅是支持Markdown语法的，Markdown语法也是很好学的，语法形式简单易懂。Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

git作为版本控制工具，采用了分布式版本库的作法，不需要服务器端软件，就可以运作版本控制，使得源代码的发布和交流极其方便。同时，使用几条简单的hexo指令和git指令就可以从本地提交自己的项目、文档、源代码等到远端，这是非常迅速的。

github远端这个就不用多说了吧，可以在远端看自己的记录。
